Div is a block element, will take up the entire width available,

Span is an Inline element, will only take up what is needed

div
span

We can add a height and width to a block element to adjust its place

With an inline element, the width and height wouldnt apply .. unless you add the display property of block

div
width="150px;" height = 150px; div is already a block so no display property needed
span
width= 150px; height =150px; display:block;
div
span

These elements have now been set to inline, and can not be adjusted using height and width




div
width="150px;" height = 150px;
span
width="150px;" height = 150px;

These elements have now been set to inline-block and can have width and height applied
